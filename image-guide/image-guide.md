# SafeD 웹사이트 이미지 제작 가이드

## 공통 스타일 가이드

- **색상 톤**: 블루 계열 (#2563EB), 다크 네이비 (#1C1C1E), 화이트
- **UI 캡처 스타일**: 실제 SafeD 화면을 깔끔하게 크롭, 데이터가 채워진 상태

- **아이콘**: 컬러 배경 원형 안에 라인 아이콘
- **비율**: UI 캡처는 4:3 또는 16:10, 히어로/슬라이드는 16:9
