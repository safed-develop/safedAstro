## 캡처해야 할 SafeD 화면 목록

SafeD 시스템(safed.co.kr)에서 캡처가 필요한 화면:

| # | 캡처 화면 | 용도 | 파일명 |
|---|----------|------|--------|
| 1 | **메인 대시보드** (현장 목록, 이행률 그래프) | 메인 핀스크롤 카드1, 대시보드 섹션 | `main_img_01.png`, `main03.png`, `dboard.png` |
| 2 | **위험성평가 작성 화면** (공종 선택 → 위험요인 매칭) | 메인 핀스크롤 카드2 | `main_img_02.png` |
| 3 | **다국어 번역 결과 화면** (베트남어/중국어 번역) | 메인 핀스크롤 카드5 | `main_img_03.png` |
| 4 | **4대 의무사항 매핑 화면** 또는 근거자료 생성 | 메인 핀스크롤 카드3 | `feature05.jpg` |
| 5 | **모바일 앱 메인 화면** | 메인 시작하기 섹션 | 앱 프리뷰 이미지 |

### 캡처 팁
- 데모/테스트 데이터가 충분히 채워진 상태에서 캡처
- 브라우저 전체화면 (주소바 제거)
- 해상도: 최소 1920x1080 (Retina 대응 시 2x)
- 민감한 데이터 (이름, 연락처 등) 블러 처리
